# App images

Add the required image assets here. The app expects:

- `icon.png` – app icon
- `partial-react-logo.png` – home screen header (e.g. 290×178)
- `react-logo.png` – explore screen (e.g. 100×100)
- `splash-icon.png` – splash screen
- `favicon.png` – web favicon
- `android-icon-foreground.png`, `android-icon-background.png`, `android-icon-monochrome.png` – Android adaptive icon

You can copy these from a fresh Expo template (`npx create-expo-app@latest temp --template tabs` then copy `assets` from `temp`) or use your own assets.
